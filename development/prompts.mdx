LangBear introduces a web user interface that enables easy creation and management of prompts.
Users can conveniently draft, alter, and organize prompts on the platform. This practical feature
streamlines the prompt development process, ensuring a user-friendly experience.

## Adding prompts to LangBear app

### Common fields

- **key** - It is a unique identifier for the prompt.
- **description** - It is a short description of the prompt.
- **tags** - It is a list of tags for the prompt.
- **type** - It is a type of the prompt. It can be `llm` or `chat`.

## Getting prompts from LangBear app

Don't forget to initialize [LangBear client](/development/client) before using it.

### Fetch a prompt

The `get_prompt_template` method returns a prompt template related to the specified key.
If the type is `llm`, the return type will be `PromptTemplate`. Conversely, for the
`chat` type, the return type will be `ChatPromptTemplate`.

<CodeGroup>

```python python
prompt = lb.get_prompt_template("company-name-generation")
```
